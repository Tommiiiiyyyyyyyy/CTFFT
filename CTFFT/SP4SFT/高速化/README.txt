検証用のために，アルゴリズムを高速化しました．
基本的には，define_param.m 内のパラメータを変えると結果が変わります．
・ベクトルの長さ n を 2^N で与える N．
・window function のパラメータである delta を与える c．
・d を与える d_c．
・L を与える l．
・ベクトルの分割数である B と，d*k を与える k．
特に重要なパラメータは，B, k, l, d_c です．
おおよそ，B≒2^(N/2)，k≒B/2~4 程度です．

変換につかうベクトルのデータは，make_sample_data_1.m の中身を書き換えてください．
SFT_main.m で実行することができます．
